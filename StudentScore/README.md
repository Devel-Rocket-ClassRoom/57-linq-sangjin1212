# 학생 성적 분석

학생들의 성적 데이터를 분석하는 프로그램을 작성함.

아래 주어진 `Student` 클래스와 학생 목록을 사용하여 다양한 분석 쿼리를 구현함.

**Student 클래스**

- `Name`: 이름 (string)
- `Subject`: 과목 (string)
- `Score`: 점수 (int)

**학생 목록**

| 이름 | 과목 | 점수 |
|------|------|------|
| 김철수 | 수학 | 85 |
| 김철수 | 영어 | 78 |
| 이영희 | 수학 | 92 |
| 이영희 | 영어 | 88 |
| 박민수 | 수학 | 76 |
| 박민수 | 영어 | 82 |
| 정지은 | 수학 | 95 |
| 정지은 | 영어 | 91 |

**구현할 쿼리**

1. 점수가 85점 이상인 성적 출력
2. 수학 과목의 모든 성적 출력
3. 점수 내림차순으로 정렬하여 출력
4. 전체 평균 점수 계산
5. 수학 과목 최고 점수와 최저 점수 출력
6. 영어 과목에서 90점 이상인 학생이 있는지 확인
7. 모든 학생이 70점 이상인지 확인
8. 학생 이름 목록 (중복 제거) 출력
9. 수학 과목에서 가장 높은 점수를 받은 학생 이름 출력
10. 과목별로 점수 내림차순, 같은 점수면 이름순으로 정렬
